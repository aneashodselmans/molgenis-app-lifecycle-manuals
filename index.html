How to add core variables for cohort

Step 1
Open the "Core variable" table
Step 2
To add a core variable, click on the green +, left in the header of the table.
Step 3
Fill out the form.
Step 4
Open the "Menu" table.
In this example we will be adding a variable to "maternal characteristics” item.
Step 5
Click on the edit row of the maternal characteristics row, which is the white button on the beginning of the row.
Step 6
To add your core variable to the menu item, select a value in the "variables" attribute and save the changes.
Please note that the dropdown in the variables starts working as soon as you start typing in the text box.
Step 7
You can select as many values as you want. To close the dropdown: click on the white part of the pop up next to the “+” buttons.
IMPORTANT: Do not click in the grey area outside the modal, this causes your pop up to close.
The maternal characteristics and socio demographic characteristics are already related in the menu. You can select your variables in the socio-demographic characteristics and they will automatically be stored in socio-demographic characteristics, which have as parent, the maternal characteristics.

How to add/update source variables in the catalogue (in data explorer)

Step 1
Open the source variable table of your cohort:
In this example the DNBC cohort is selected.
Step 2
Click on the green + sign to add a row and click on the edit button in front of the row.
IMPORTANT: Make sure that when using both editing options (excel upload/molgenis edit), you should alter both, otherwise when updating your changes will become undone.
In this example edit row is clicked. When clicking "add row", the form will be empty.
Step 3
Now you can fill out the form and click on save.
IMPORTANT: Do not click in the grey area outside the modal, this causes your pop up to close.
Your variable is now added/updated. Nightly a mapping will be applied which will put your updated variable in the main source variable table, this means your variable will not instantly be updated in the main table.

How to upload source variables for cohort

Step 1
Create an excel file with as tabname: LifeCycle_*here your cohort name*_SourceVariables.
Please note the casing, L, C, S and V are capitals.
Use the following format for your table:

cohort variable datatype values unit collectionType description dateOfUpdate dependencies
The identifier of the cohort The name of the variable The type of the value, choose from Choose from these The possible values for the variable Which unit is the variable in? Choose from these How was the information collected? Describe of the variable Date that the variable was last updated for harmonization Description of dependencies of the variable on other variables

Step 2
Fill the file with your data.
Step 3
Go to the importer.
Step 4
Click on "Select a file..." and select your file. Then click the next button on the bottom right.

Possible errors

Solution:

Probably you made a mistake in naming the tab in excel. Note this is case sensitive and should be in the following format: LifeCycle_*cohort name*_SourceVariables. Click here for a list of all possible cohorts. This name is case sensitive as well.
Step 4
Select: Add entities / update existing. Click the next button on the bottom right again. The package step appears, since you are only uploading data you can ignore this and press next again. If you provided all columnheaders in your file, all fields will be green and you can press next again.

Possible errors

Solution:

You provided a date that is not formatted in a molgenis supported date format. Please make sure you use this format: yyyy-mm-dd.

Solution:

Description is a required column. Please provide a description for all rows.

Solution:

The cohort name you provided is not a valid cohort name. Please select one from this list. Again, the data is case sensitive, so make sure all casing is correct. Look at the "ID" column for this value.

How to add/update harmonization in the catalogue (in data explorer)

Step 1
Open the haronizations table of your cohort:
In this example the DNBC cohort is selected.
Step 2
Click on the green + sign to add a row and click on the edit button in front of the row.
IMPORTANT: Make sure that when using both editing options (excel upload/molgenis edit), you should alter both, otherwise when updating your changes will become undone.
In this example add row is clicked. When clicking "edit row", the form will be filled with content.
Step 3
Now you can fill out the form and click on create.
IMPORTANT: Do not click in the grey area outside the modal, this causes your pop up to close.
